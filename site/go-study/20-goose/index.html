Goose

  • Goose is a database migration tool, both CLI and a library.
  • Manage your database schema by creating incremental SQL changes or Go functions.

Installation

Homebrew:

brew install goose

Go Install: this will install the goose library to your $GOPATH/bin directory.

go install github.com/pressly/goose/cmd/goose@latest

Usage

Create a new migration

goose create add_some_column sql
$ Created new file: 20170506082420_add_some_column.sql

goose -s create add_some_column sql
$ Created new file with sequential numbering: 00001_add_some_column.sql

goose create fetch_user_data go
$ Created new go file: 20170506082421_fetch_user_data.go

Apply all available migrations

goose up

Migrate up to a specific version

goose up-to 20170506082421

Migrate up a single migration from the current version

goose up-by-one

Rollback a single migration from the current version

goose down

Rollback migration to a specific version

goose down-to 20170506082421

Rollback all migrations

goose down-to 0

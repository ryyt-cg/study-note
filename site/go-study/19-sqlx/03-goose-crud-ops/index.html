CRUD Operations with goose

Overview

This example is to build a simple CRUD application using SQLX, SQLite3, and some best practices.

Project Structure

├── README.md
├── cmd
│   └── server.go
├── go.mod
├── go.sum
├── goose-crud.sqlite
├── migrations
│   └── 20241219003415_add_user.sql
└── pkg
    ├── db
    │   └── sqlite.go
    └── repository
        ├── author.go
        ├── author_repository.go
        ├── book.go
        └── book_repository.go

Step to run application

  1. Use goose tool create database and tables
goose -dir migrations sqlite3 goose-crud.sqlite up
  1. Run the application
go run cmd/server.go
